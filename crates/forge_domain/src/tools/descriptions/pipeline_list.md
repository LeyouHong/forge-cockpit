Lists the user's saved pipelines — reusable YAML workflows (DAGs of AI-agent and shell nodes) built in the Forge web UI or CLI. Returns each pipeline's file name, workflow name, description, declared inputs (with defaults), and node ids. ALWAYS check this list when the user asks for a task that an existing pipeline might automate (e.g. "review this PR" when a pr-review pipeline exists) — running the pipeline is preferred over doing the equivalent work manually. Also call it before pipeline_run to discover the exact file name and required inputs.
